高级设置

    Twitter API

  • 您还没有启用 Sub-Consumer.
  • 由于 Twitter 限制了单位时间内 API 的使用额度, PREFiX 可能会因为超出使用限额而无法正常工作. 启用 Sub-Consumer 后, 您的使用额度将会变为原来的 2 倍, 可以改善您的使用体验.

    内部处理规则

  • 可以改善性能, 但仅建议当关注的人较多时启用.

    Timeline

  • 每页加载的推文数量
  • 当 Timeline 上的消息更新非常快时, 建议使用较大的加载数量, 但性能可能会稍有下降.
